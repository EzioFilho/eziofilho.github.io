Eden Project

Eden Project é o projeto mais ambicioso da 2GeekGames, nele o jogador poderá viajar por diversas eras na história da humanidade e poderá escolher alterá-las ou não, podendo até mesmo descobrir o que deu fim aos dinossauros. Eden Project foi desenvolvido na Engine de Jogos Unreal Engine 4, e mescla a utilização de c++ e blueprints. Link do Executável: Eden Project

Donkey Treasures

DT é um jogo que relembra a mecânica de um clássico arcade, desenvolvido para dispositivos Mobile pela minha empresa, 2GeekGames. O jogo foi desenvolvido na plataforma Androide Studio na linguagem Java.

Claustrofobia

Claustrofobia é um jogo de Terror e Puzzles, ambientado em Recife, uma cidade Brasileira. O jogo utiliza do foclore e lendas urbanas Brasileiras. Claustrofobia foi desenvolvido na Engine de jogos Unity e escrito em c#. Link para o executável: Claustrofobia

FisioVet

Fisiovet é um jogo educacional de perguntas e respostas com diversos níveis diferentes voltados para fisiologia veterinária. Fisiovet é feito na Engine de Jogos Unity, escrito em c#.

Animap

Animap foi um aplicatio desenvolvido para dispositivos mobile da Apple, no curso de programação em swift disponibilizado pelo projeto HackATruck

HighNoon

HighNoon é um jogo multiplayr em que desenvolvi em Python, com a biblioteca pygame. As mecânicas principais são simplesmente apreendidas mas dependem de treino para masteriza-las.
